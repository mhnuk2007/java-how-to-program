## 📚 Key Topics

### 🔰 Core Java
- Variables, data types, operators, expressions  
- Control statements (if, switch, loops)  
- Methods, recursion, arrays, strings  

### 🧱 Object-Oriented Programming
- Classes & objects, encapsulation  
- Inheritance, polymorphism  
- Interfaces & abstract classes  

### ⚠️ Exception Handling
- try-catch-finally, custom exceptions  
- Chained & rethrown exceptions  
- Try-with-resources  

### 🖥️ GUI Programming (Swing)
- Labels, buttons, text fields, checkboxes, radio buttons, lists  
- Event handling & interactive GUI applications  

### 🗃️ File Handling & I/O
- Byte & character streams  
- Serialization & deserialization  
- NIO basics, file reading & writing  

### 🧺 Collections & Generics
- List, Set, Map, sorting & searching  
- Generic classes & methods  

### 🧠 Lambdas & Streams (Java 8+)
- Functional interfaces  
- Lambda expressions  
- Stream API (map, filter, reduce)  
- Predicate, Function, Consumer, Supplier  

### 🧮 Algorithms & Recursion
- Searching & sorting  
- Recursive problem solving  
- Algorithmic exercises
